# SHREDDED

本關卡為 Google CTF Beginner's Quest 的 Miscellaneous 挑戰。目標是將 100 張被垂直切碎的手寫筆記圖片重新拼接，以還原出原始的手寫筆記並取得 Flag。

## 解題思維與演算法

1. **邊緣相似度計算**：
   - 提取每張碎紙圖片的最左側與最右側像素邊緣。
   - 計算任兩張圖片相鄰時，邊緣像素的差異度（即 L1 距離）。

2. **鏈路重建 (Kruskal-based TSP)**：
   - 使用貪婪演算法（Greedy）與並查集（Union-Find），在避免局部成環且保證每個節點度數限制（最多一入一出）的前提下，依序連接最相似的邊緣，將 100 張紙條串聯成單一鏈條。

3. **環狀位移與旋轉**：
   - 將寬度為 52 像素的全白邊界圖片（`54164269.png`）移至最左側作為起始。
   - 最後將拼接完成的影像逆時針旋轉 90 度以利直觀閱讀。
